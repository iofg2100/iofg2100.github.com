Octopressでブログを作った

Octopress と GitHub Pages でブログを作りました。

Octopress は、

  • Markdownで記事を記述できる
  • 静的記事も作れる
  • サイトを一度ローカルで生成する
  • GitHub Pages にコマンド一発でデプロイできる

ということで導入を決定しました。

公式サイトのdocumentationで導入法が詳しく説明されているほか、様々なサイトに解説記事があります。
